## 🚀 Portfolio Projects

### 🐳 [Docker → Kubernetes CI/CD Deployment Pipeline](https://github.com/Rabaanee/docker-k8s-deployment-pipeline)

A full end-to-end CI/CD pipeline taking a containerised Python app from code commit to Kubernetes deployment with zero manual steps.

**What's inside:**
- Multi-stage Docker build (tests run inside the build; ~217MB production image)
- Non-root container security (`appuser`)
- Kubernetes deployment with rolling updates, liveness/readiness probes, and HPA
- Custom Helm chart with per-environment values (dev / staging / prod)
- GitHub Actions CI: lint → pytest → Trivy security scan → Docker build → Helm lint
- GitHub Actions CD: build → push to Docker Hub → deploy via Helm
- Branch protection rules enforcing all checks before merge

`Docker` `Kubernetes` `Helm` `GitHub Actions` `Trivy` `Python` `DevSecOps`
